# Desenvolvimento de Aplicativo - 2021/1

## Configuração de variáveis

Crie um arquivo chamado keys, dentro da pasta config desse formato:

```js
export default {
  firebaseConfig: {
    apiKey: '',
    authDomain: '',
    projectId: '',
    storageBucket: '',
    messagingSenderId: '',
    appId: '',
  },
};

```
